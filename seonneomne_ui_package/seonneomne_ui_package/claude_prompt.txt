첨부 이미지와 아래 파일을 기준으로 UI를 구현해줘.
목표는 분위기 재창작이 아니라, 첨부 이미지의 화면 비율/여백/객체 위치/크기/텍스트 줄바꿈을 최대한 그대로 재현하는 것이다.

기준 캔버스는 390px × 694px, 9:16 화면이다.
이미지 원본은 941px × 1672px이고, 390px 폭으로 축소한 pixel-perfect 기준이다.
첫 구현은 반응형보다 고정형 재현을 우선한다.

중요 규칙:
1. 전체 화면 비율을 바꾸지 말 것.
2. 카드 위치, 카드 크기, 버튼 위치를 임의로 조정하지 말 것.
3. 텍스트 줄바꿈을 임의로 바꾸지 말 것.
4. border-radius, padding, gap을 임의로 최적화하지 말 것.
5. 손그림 로고, 배경 워터마크, 아이콘은 가능하면 SVG/PNG asset으로 사용한다.
6. CSS로 억지로 비슷하게 그리기보다 asset을 배치하는 쪽을 우선한다.
7. 실제 앱 구현용이면 status bar는 제거하고, 프로토타입 캡처용이면 status bar를 유지한다.
8. 구현 후 브라우저 캡처를 원본 이미지와 비교해서 다른 부분만 수정한다.

전달 파일:
- UI_SPEC.md: 기준 치수와 레이아웃 명세
- src/App.jsx: React 구조 예시
- src/seonneomne.css: pixel-first CSS 초안
